> [!NOTE]

> [!IMPORTANT]
> Features ✨:
> 1. Obfuscation with two security levels (Standard/Advanced)
> 2. Bytecode compilation to .pyc files
> 3. Automatic backups before encryption
> 4. Easy recovery from backups via number selection
> 5. Built-in dependency installation
> 6. Clean interface with loading animations
> 7. Works on Android (Termux) and other platforms
